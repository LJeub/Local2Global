local2global.example.add_noise¶

add_noise(alignment_problem: local2global.utils.AlignmentProblem, noise_level=1, scales=None)[source]¶

Add random normally-distributed noise to each point in each patch

Parameters

  • alignment_problem (AlignmentProblem) – Alignment problem to be transformed

  • noise_level – Standard deviation of noise

  • scales – (optional) list of scales for each patch (noise for patch is multiplied by corresponding scale)
