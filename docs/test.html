The local2global package includes tests that check reconstruction performance on synthetic data. The test use pytest and are contained in local2global.test_local2global.

To run the tests use

python -m local2global.test_local2global [args]

where any command-line arguments are passed along to pytest. For details of the test see test_local2global.
